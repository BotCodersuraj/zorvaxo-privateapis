About Bullet Storm

Bullet Storm is an addictive 2D shooting game. Move with arrow keys and shoot with spacebar. Dodge enemies, collect power-ups and survive as long as you can!

  • Top-down 2D shooter
  • Multiple enemy types & levels
  • Power-ups and achievements
  • Leaderboard to challenge friends
